# Digit Recognition 🔢

[![Live Demo](https://img.shields.io/badge/🤗%20Hugging%20Face-Live%20Demo-blue)](https://huggingface.co/spaces/ZMIMS/Digit-recognition)

Handwritten digits recognition model trained using MNIST dataset. Hand draw a number and see what the model thinks about it.

## Technology Stack
- TensorFlow / Keras
- Gradio
- Python 3.13

## Directory
- `app.py` - Gradio application
- `handwritten.keras` - Model used to predict numbers (trained on MNIST dataset, 98% accuracy)
- `requirements.txt`
